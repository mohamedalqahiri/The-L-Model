# Contributing to The L-Exponential Model

We welcome global independent researchers, heterodox economists, blockchain developers, and system engineers to collaborate on developing this framework from scarcity to absolute abundance.

## How to Contribute
1. **Review Foundations:** Read `ECONOMIC-THEORY.md` and `WHITEPAPER.md` before making architectural proposals.
2. **Submit Issues:** Report mathematical anomalies, simulation calibration improvements, or systemic edge cases via the Issues tab.
3. **Pull Requests:** * Fork the repository.
   * Create a feature branch reflecting specific optimization nodes (e.g., `feature/optimize-sync-function`).
   * Submit a PR detailing the physical or algorithmic impact of your change.

## Academic and Open-Access Integrity
By contributing to this repository, you agree that your work will be permanently bound under the **Creative Commons CC BY-NC-SA** license framework to prevent commercial capture and ensure free access for all humanity.
